# Gaussian Blur Tool

一个纯前端高斯模糊图片工具，支持上传、拖拽、粘贴图片、实时预览、PNG/JPEG 导出。

## 部署到 GitHub Pages

1. 在 GitHub 新建一个公开仓库，例如 `gaussian-blur-tool`。
2. 上传本文件夹里的所有文件到仓库根目录。
3. 打开仓库的 `Settings`。
4. 进入 `Pages`。
5. 在 `Build and deployment` 中选择 `Deploy from a branch`。
6. Branch 选择 `main`，Folder 选择 `/root`。
7. 保存后等待 GitHub 生成公开网址。

生成的网址通常类似：

```text
https://你的用户名.github.io/gaussian-blur-tool/
```
